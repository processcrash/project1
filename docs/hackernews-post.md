**Title:** Show HN: I built an AI code reviewer that catches bugs before production (self-hosted, free tier available)

**Body:**

Hey HN!

I've been working on CodeSentinel - an AI-powered code review platform that automatically analyzes code changes and identifies bugs, security vulnerabilities, and code quality issues before they reach production.

**How it works:**
1. Submit code or connect your GitHub repo
2. AI analyzes the code using GPT-4/Claude
3. Get detailed feedback with line-by-line issues, severity levels, and fix suggestions

**Features:**
- 1-10 quality score with detailed breakdown
- Detects bugs, security issues, performance problems
- Multi-language: JS, TS, Python, Go, Rust, Java, C++
- GitHub integration (automatic PR reviews)
- Team collaboration
- Slack notifications
- Free tier (100 reviews/month)

**Why this instead of GitHub Copilot?**
Copilot helps you WRITE code. CodeSentinel helps you REVIEW code. Different use cases, and we do the review part really well.

**Tech:** React + Express + TypeScript + TailwindCSS
